LINEAR SEARCH- Level-1

What is a Linear Search?

The Linear Search algorithm searches through an array and returns the index of the value it searches for.

Algorithm: Steps

Step 1: Set i to 1

Step 2: If i > n, then jump to step 7

Step 3: If A[i] == x then jump to step 6

Step 4: Set i = i + 1

Step 5: Go to step 2

Step 6: Print "Element x found at index i" and jump to step 8

Step 7: Print "Element not found"

Step 8: Exit

Algorithm: Linear Search (Array A, Value x)

Manual Run-Through:

We will search for value 11 in the array: [12, 8, 9, 11, 5, 11]

Step 1: Start at index 0 → 12 ≠ 11

Step 2: Check index 1 → 8 ≠ 11

Step 3: Check index 2 → 9 ≠ 11

Step 4: Check index 3 → 11 = 11 ✅

Result: Value 11 is found at index 3

How Linear Search Works:

  • Go through the array value by value from the start.
  • Compare each value to check if it is equal to the value we are looking for.
  • If the value is found, return the index of that value.
  • If the end of the array is reached and the value is not found, return -1.

Linear Search Implementation:

  • An array with values to search through.
  • A target value to search for.
  • A loop that goes through the array from start to end.
  • An if-statement that compares the current value with the target value and returns the index if it matches.
  • After the loop, return -1 to indicate not found.

Linear Search Time Complexity:

Linear Search has a time complexity of O(n) where n is the number of elements in the array.

  • If the value is found at the first position → only 1 comparison is needed.
  • If the value is found at the end or not found at all → n comparisons are needed.
