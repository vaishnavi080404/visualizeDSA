What is a Stack?

A stack is a data structure that stores elements in a LIFO (Last In First Out) manner. Think of it like a stack of pancakes â€“ you add and remove from the top.

  • Push: Adds a new element on top of the stack.
  • Pop: Removes and returns the top element.
  • Peek: Returns the top element without removing it.
  • isEmpty: Checks if the stack is empty.
  • Size: Returns the number of elements in the stack.

Visual Example:

Imagine pushing values like 2, 1, 4, and 4 on a stack. The stack would look like this (top at bottom):

Stack Implementation using Arrays

  • Memory Efficient: No need to store addresses.
  • Simple and easy to implement.
  • However, arrays are of fixed size.

Example Stack: [1, 3, 4, 1]

Stack Implementation using Linked Lists

  • Dynamic size: Can grow and shrink as needed.
  • Each element stores a pointer to the next.
  • Requires extra memory for pointers.

Applications of Stack:

  • Undo functionality in editors
  • Backtracking in algorithms
  • Expression evaluation
  • Depth-First Search (DFS)
