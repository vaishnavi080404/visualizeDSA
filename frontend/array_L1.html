Understanding Array

💡 What is an Array?

An Array is a collection of items of the same variable type stored at contiguous memory locations. It is one of the most popular and simple data structures used in programming. Each element in an array is accessed by its index, which in most programming languages starts from 0.

🤔 Why Use Arrays?

Assume there is a class of five students and if we have to keep records of their marks, we can declare five variables individually. But if the number of students becomes very large, it would be challenging to manipulate and maintain data individually. Arrays solve this by storing multiple values under a single variable name, allowing efficient access, traversal, and manipulation.

🔄 How Does an Array Work?

An array allocates a block of contiguous memory at declaration time. Each element occupies a fixed-size slot and can be accessed directly using its index in O(1) time. This direct addressing makes arrays extremely efficient for read-heavy operations.

Access (by index)
O(1)
Search (linear)
O(n)

Basic Terminologies

Array Element
Each item stored in an array. Elements are of the same data type and accessed by their index.
Array Index
The position of an element. Indexes start from 0 in most languages (0-based indexing).
Array Size / Length
The total number of elements an array can hold. Fixed at declaration time for static arrays.
Contiguous Memory
All array elements are stored in adjacent memory locations, enabling O(1) random access.

Types of Arrays

Arrays can be classified in two ways: on the basis of Size and on the basis of Dimensions.

1
Fixed Size Array (Static)
▼

A fixed size array has a predefined capacity that cannot be changed after declaration. Memory is allocated at compile time (stack memory). If you declare a larger size than needed, memory is wasted; if smaller, you can't store all elements.

  • Size must be known at compile time
  • Memory allocated on the stack (fast)
  • Cannot grow or shrink dynamically
  • Out-of-bounds access leads to undefined behavior

Java
Python
JavaScript
C
// Fixed size array (compile-time allocation)

// Dynamic fixed array (run-time allocation)

dynArr[0] = 10;
free(dynArr);
C++
// Compile-time fixed array
int arr[5] = { 10, 20, 30, 40, 50 };

// Runtime dynamic fixed array
int *dynArr = new int[5];
dynArr[0] = 10;
delete[] dynArr;
Java
// Fixed size array in Java
int[] arr = new int[5];
arr[0] = 10; arr[1] = 20;

// Or inline
int[] arr2 = { 10, 20, 30, 40, 50 };

// ArrayIndexOutOfBoundsException if arr[5] accessed
Python
# Fixed-type array using array module

print(arr[0])  # 10

# NumPy fixed-size array

np_arr = np.array([10, 20, 30, 40, 50])
JavaScript
// TypedArray provides fixed-size typed storage
const arr = new Int32Array([10, 20, 30, 40, 50]);

2
Dynamic Size Array
▼

A dynamic size array can grow and shrink as needed during program execution. Memory is allocated on the heap. No need to predefine exact size — elements can be added or removed freely.

  • Size grows or shrinks at runtime
  • Memory allocated on heap (flexible)
  • Used in most modern programming via built-in types
  • Slightly slower than static due to reallocation overhead

➕ Insertion

Adding a new element into the array at a specific position.

1
Insert at Beginning
▼

To insert at the beginning, shift all existing elements one position to the right, then place the new element at index 0.

Time Complexity
O(n)
Space Complexity
O(1)
C
C++
Java
Python
JavaScript
C
void insertAtBeginning(int arr[], int *n, int val, int cap) {
    if (*n == cap) { printf("Array Full\n"); return; }
    // Shift all elements to the right
    for (int i = *n; i > 0; i--)
        arr[i] = arr[i - 1];
    arr[0] = val;
    (*n)++;
}
C++

// With raw array
void insertAtBeginning(int arr[], int& n, int val) {
    for (int i = n; i > 0; i--)
        arr[i] = arr[i-1];
    arr[0] = val; n++;
}
Java
int[] insertAtBeginning(int[] arr, int val) {
    int[] newArr = new int[arr.length + 1];
    newArr[0] = val;
    for (int i = 0; i < arr.length; i++)
        newArr[i + 1] = arr[i];
    return newArr;
}

Removing an element from the array. Elements are shifted to fill the gap.

1
Delete from Beginning
▼

Remove the first element by shifting all other elements one step to the left. Decrements array size.

C
void deleteFromBeginning(int arr[], int *n) {

    for (int i = 0; i < *n - 1; i++)

🔍 Searching

Finding an element in the array. Two main approaches: Linear Search and Binary Search.

1
Linear Search
▼

Check each element one by one from the start until the target is found or the array ends. Works on unsorted arrays.

Best Case
O(1)
Worst Case
O(n)
Space
O(1)
C
C++
Java
Python
JavaScript
C
int linearSearch(int arr[], int n, int key) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == key)
            return i;  // return index
    }
    return -1;  // not found
}
C++
int linearSearch(vector<int>& arr, int key) {
    for (int i = 0; i < arr.size(); i++)
        if (arr[i] == key) return i;
    return -1;
}
Java
int linearSearch(int[] arr, int key) {
    for (int i = 0; i < arr.length; i++)
        if (arr[i] == key) return i;
    return -1;
}
Python
def linear_search(arr, key):
    for i, val in enumerate(arr):
        if val == key:
            return i
    return -1
JavaScript
function linearSearch(arr, key) {
    for (let i = 0; i < arr.length; i++)
        if (arr[i] === key) return i;
    return -1;
}
2
Binary Search (Sorted Array)
▼

Binary search works on sorted arrays. It repeatedly divides the search interval in half. Compare the middle element with the target; if equal, found. If target is greater, search right half; else left half.

Best Case
O(1)
Worst Case
O(log n)
Space
O(1)
C
C++
Java
Python
JavaScript
C
int binarySearch(int arr[], int n, int key) {
    int low = 0, high = n - 1;
    while (low <= high) {
        int mid = low + (high - low) / 2;
        if (arr[mid] == key) return mid;
        else if (arr[mid] < key) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}
C++
int binarySearch(vector<int>& arr, int key) {
    int low = 0, high = arr.size() - 1;
    while (low <= high) {
        int mid = low + (high - low) / 2;
        if (arr[mid] == key) return mid;
        else if (arr[mid] < key) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}
Java
int binarySearch(int[] arr, int key) {
    int low = 0, high = arr.length - 1;
    while (low <= high) {
        int mid = low + (high - low) / 2;
        if (arr[mid] == key) return mid;
        else if (arr[mid] < key) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}
Python
def binary_search(arr, key):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == key:
            return mid
        elif arr[mid] < key:
            low = mid + 1
        else:
            high = mid - 1
    return -1
JavaScript
function binarySearch(arr, key) {
    let low = 0, high = arr.length - 1;
    while (low <= high) {
        const mid = Math.floor((low + high) / 2);
        if (arr[mid] === key) return mid;
        else if (arr[mid] < key) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}
