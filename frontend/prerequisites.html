Variables store data in named memory locations. Each variable has a type, name, and value.
A variable in C is a named piece of memory which is used to store data and access it whenever required. It allows us to use the memory without having to memorize the exact memory address.

Data Types in C:

  • int: Integer values (4 bytes) - stores whole numbers like 10, -5, 0
  • float: Floating-point (4 bytes) - decimal values like 3.14, -0.5
  • double: Double precision (8 bytes) - larger decimals with more precision
  • char: Character (1 byte) - single character like 'A', 'z', '5'
#include <stdio.h>

int main() {
    // Variable declaration and initialization
    int age = 20;
    float height = 5.7;
    char grade = 'A';

    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Grade: %c\n", grade);

    return 0;
}
Output
Age: 20
Height: 5.7
Grade: A

Variable Naming Rules:

  • Must start with letter or underscore (_)
  • Can contain letters, digits, underscores only
  • Cannot use C keywords (int, return, for, while, etc.)
  • Case-sensitive (age and Age are different variables)
  • No spaces allowed in variable names

Loops repeat code multiple times. Each iteration executes the block until the condition is false.
Loops in C programming are used to repeat a block of code until the specified condition is met. They allow programmers to execute statements multiple times without writing code repeatedly.

Types of Loops:

  • for loop: Entry-controlled loop, use when number of iterations is known
  • while loop: Entry-controlled loop, condition checked before execution
  • do-while loop: Exit-controlled loop, executes at least once

// Dynamic memory allocation with malloc
#include <stdio.h>
#include <stdlib.h>

int main() {
    int *arr = (int*)malloc(5 * sizeof(int));

    if (arr == NULL) {
        printf("Memory allocation failed");
        return 1;
    }

    // Store values
    for (int i = 0; i < 5; i++) {
        arr[i] = i + 1;
    }

    // Print values
    for (int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }

    free(arr);  // Release memory
    return 0;
}
Output
1 2 3 4 5

ES6 introduced class syntax for creating objects. JavaScript uses prototypal inheritance underneath. 
class Student {
    constructor(name, roll) {
        this.name = name;
        this.roll = roll;
        this.grades = [];
    }
    
    addGrade(grade) {
        this.grades.push(grade);
    }
    
    get average() {
        const sum = this.grades.reduce((a, b) => a + b, 0);
        return sum / this.grades.length;
    }
    
    static compareStudents(s1, s2) {
        return s1.average - s2.average;
    }
}

const student = new Student("John", 101);
student.addGrade(85);
console.log(student.average);
Output
85
