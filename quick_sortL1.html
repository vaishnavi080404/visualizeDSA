QUICK SORT- Level-1

What is a Quicksort?

As the name suggests, Quicksort is one of the fastest sorting algorithms.

The Quicksort algorithm takes an array of values, chooses one of the values as the 'pivot' element, and moves the other values so that lower values are on the left of the pivot element, and higher values are on the right of it.

Algorithm: Steps

Step 1: Choose a pivot element from the array.

Step 2: Partition the array into two sub-arrays: one with values less than the pivot, and the other with values greater than the pivot.

Step 3: Recursively apply Quick Sort to the sub-arrays.

Step 4: Continue until the base case (sub-array with 1 or 0 elements) is reached.

Step 5: Return the sorted array.

Algorithm: Quick Sort (Array A, Low, High)

Manual Run-Through:

We will sort the array: [11, 9, 12, 7, 3] using the Quick Sort algorithm.

Step 1: Choose the last value as the pivot (3).

Step 2: Rearrange the array: [3, 9, 12, 7, 11]

Step 3: The pivot 3 is in the correct position. Now, apply Quick Sort on the sub-arrays to the left and right of 3.

Step 4: Sort sub-array [9, 12, 7, 11] with pivot 11. New array: [3, 9, 7, 11, 12]

Step 5: Sort sub-array [9, 7] with pivot 7. Final sorted array: [3, 7, 9, 11, 12]

How Quick Sort Works:

  • Quick Sort works by selecting a pivot element from the array.
  • The array is then partitioned into two parts: values less than the pivot and values greater than the pivot.
  • Each of these sub-arrays is sorted recursively.
  • When the base case is reached, the array is sorted.

Quick Sort Implementation:

  • An array with values to sort.
  • A method to partition the array around a pivot element.
  • Recursively apply Quick Sort to the sub-arrays.

Quick Sort Time Complexity:

Quick Sort has an average time complexity of O(n log n), where n is the number of elements in the array.

  • In the best and average cases, the pivot divides the array evenly.
  • The worst-case time complexity is O(n²), which occurs when the pivot is always the smallest or largest element.
