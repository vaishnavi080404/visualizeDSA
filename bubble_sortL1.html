BUBBLE SORT- Level-1

What is a Bubble Sort?

Bubble Sort is an algorithm that sorts an array from the lowest value to the highest value.

Algorithm: Steps

Step 1: Repeat from i = 0 to n-1

Step 2: For each pass, repeat from j = 0 to n-i-1

Step 3: If A[j] > A[j+1], then swap them

Step 4: Continue until the array is sorted

Step 5: Exit

Algorithm: Bubble Sort (Array A)

 
bubbleSort(array)
  for i <- 1 to sizeOfArray - 1
    for j <- 1 to sizeOfArray - 1 - i
      if leftElement > rightElement
        swap leftElement and rightElement
end bubbleSort

Manual Run-Through:

We will sort the array: [7, 12, 9, 11, 3]

Step 1: We start with an unsorted array.
[7, 12, 9, 11, 3]

Step 2: Compare 7 and 12. 7 < 12, no swap.
[7, 12, 9, 11, 3]

Step 3: Compare 12 and 9. 12 > 9, swap them.
[7, 9, 12, 11, 3]

Step 4: Compare 12 and 11. 12 > 11, swap.
[7, 9, 11, 12, 3]

Step 5: Compare 12 and 3. 12 > 3, swap.
[7, 9, 11, 3, 12]

How Bubble Sort Works:

  • Compares adjacent elements and swaps them if they are in the wrong order.
  • Largest element "bubbles" to the end of the list after each pass.
  • Repeats the process for remaining elements until the list is sorted.

Bubble Sort Implementation:

  • Use nested loops: outer loop for passes, inner loop for comparing.
  • Use a temporary variable to swap values.
  • Optimization: If no swaps in a pass, the array is already sorted.

Bubble Sort Time Complexity:

Bubble Sort has a time complexity of:

  • Best case: O(n) → already sorted, no swaps
  • Average case: O(n²)
  • Worst case: O(n²) → reverse sorted array
