Understanding Strings

💡 What is a String?

A String is a sequence of characters stored contiguously in memory. Unlike a single character, a string can hold multiple characters in order — including letters, digits, spaces, and special characters. Examples: "geeks", "GeeksforGeeks", "123Geeks", "@123 Geeks". In C/C++, strings are terminated with a null character '\0'. In Java, Python, and JavaScript, strings are immutable objects managed by the runtime.

🤔 Why Use Strings?

Strings are the fundamental building block for all text data in programming. They power user input and output, file names, network protocols, database queries, and natural language processing. Almost every real-world application processes strings in some form — from a simple login form to complex search engines.

🔄 How are Strings Stored in Memory?

Each character in a string occupies one memory slot (1 byte in ASCII, up to 4 bytes in Unicode). Characters are stored contiguously, enabling O(1) index-based access. In C, the string "hello" is stored as h e l l o \0 — the null terminator marks the end. In Java/Python/JS, strings are immutable objects with the length stored internally, giving O(1) length lookup.

Key Concepts

Null Terminator '\0'
In C/C++, every string ends with '\0' (ASCII 0). Functions like strlen scan for it to determine string length.
String Pool (Java)
Java maintains a pool of string literals. When you write "Hello" twice, Java reuses the same object from the pool to save memory.
StringBuilder
Java/C# provide StringBuilder for efficient mutable string building. Concatenating n strings with + in a loop is O(n²); StringBuilder is O(n).
ASCII vs Unicode
ASCII uses 1 byte per char (0–127). Unicode (UTF-8/16/32) supports all world scripts — some characters take 2–4 bytes, affecting length calculations.
Two-Pointer Technique
Use left and right pointers moving toward the center — efficient for palindrome check, reversal, and substring matching in O(n) time.
Pattern Matching
Algorithms like KMP and Rabin-Karp find patterns in strings efficiently — O(n+m) vs O(n×m) for brute-force search.

How Strings are Represented in Memory

Different languages handle string storage and mutability differently:

C
Declared as char arrays or pointers. Must end with '\0'. String literals via pointer are immutable; char arrays are mutable.
C++
Supports C-style arrays and the std::string class. String objects are mutable with rich built-in methods.
Java
Strings are immutable objects of the String class stored in the string pool. Use StringBuilder for mutable operations.
Python
Strings are immutable. Declared with single, double, or triple quotes. No separate char type — a single character is a string of length 1.
JavaScript
Strings are immutable primitives. Support single, double quotes, or template literals (backticks). No separate char type.
C#
Uses the string keyword — an immutable sequence of characters similar to Java. Use StringBuilder for mutations.

void insertChar(char str[], int pos, char ch) {
    int len = strlen(str);
    for(int i=len; i>=pos; i--) str[i+1] = str[i]; // shift right
    str[pos] = ch;
}

void reverseStr(char s[]) {
    int l=0, r=strlen(s)-1;
    while(l < r) {
        char t=s[l]; s[l]=s[r]; s[r]=t;
        l++; r--;
    }
}

void leftRotate(char str[], int k) {
    int n = strlen(str);
    k %= n;
    char temp[n+1];
    strncpy(temp, str+k, n-k);
    strncpy(temp+n-k, str, k);
    temp[n] = '\0';
    strcpy(str, temp);
}
char s[] = "Geeks";

printf("%s\n", s); // eksGe
C++
string leftRotate(string s, int k) {
    k %= s.size();
    return s.substr(k) + s.substr(0, k);
}

Java
String leftRotate(String s, int k) {
    k %= s.length();

🔁 10. Check for Palindrome

A string is a palindrome if it reads the same forward and backward. Examples: "racecar", "madam", "level".

1

▼
Time
O(n)

Java
Python

C
int isPalindrome(char s[]) {
    int l=0, r=strlen(s)-1;
    while(l < r) {
        if(s[l] != s[r]) return 0;
        l++; r--;
    }
    return 1;
}
printf("%s\n", isPalindrome("racecar") ? "YES" : "NO"); // YES
C++
bool isPalindrome(string s) {
    int l=0, r=s.size()-1;
    while(l < r) { if(s[l++] != s[r--]) return false; }
    return true;
}
cout << (isPalindrome("racecar") ? "YES" : "NO"); // YES
Java
boolean isPalindrome(String s) {
    int l=0, r=s.length()-1;
    while(l < r) { if(s.charAt(l++) != s.charAt(r--)) return false; }
    return true;
}
System.out.println(isPalindrome("racecar") ? "YES" : "NO"); // YES
Python
def is_palindrome(s):
    return s == s[::-1]       # Pythonic

print("YES" if is_palindrome("racecar") else "NO") # YES
JavaScript
function isPalindrome(s) {
    return s === [...s].reverse().join("");
}
console.log(isPalindrome("racecar") ? "YES" : "NO"); // YES
