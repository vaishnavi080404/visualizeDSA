Understanding Matrix

💡 What is a Matrix?

A Matrix (also called a Grid or 2D Array) is a two-dimensional data structure used to store elements arranged in rows and columns. It is mostly used in mathematical and scientific calculations. It can also be considered as an array of arrays, where each inner array has the same size — making it a rectangular grid of elements.

🤔 Why Use a Matrix?

Matrices are indispensable whenever data has a natural two-dimensional structure — game boards, images, spreadsheets, graphs (adjacency matrix), and scientific simulations all rely on them. Storing such data in a matrix gives you direct O(1) access to any cell using its row and column indices, making reads and writes extremely fast.

🔄 How Does a Matrix Work?

A matrix of size m × n has m rows and n columns. Elements are stored in contiguous memory row by row (row-major order in C/C++). Any element can be accessed in O(1) time using arr[i][j] where i is the row index and j is the column index — both starting from 0.

Access (by index)
O(1)
Traversal
O(m×n)
Search
O(m×n)
Space Complexity
O(m×n)

🔑

Key Concepts

Row-major vs Column-major
C/C++/Java store rows contiguously (row-major). Fortran/MATLAB use column-major. This affects cache performance during traversal.
Square Matrix
When m = n (rows = cols). Has a main diagonal, transpose, determinant, and inverse — used heavily in linear algebra.
Sparse Matrix
A matrix where most elements are zero. Special representations (CSR, COO) are used to save memory instead of storing every zero.
Diagonal Matrix
Non-zero elements exist only on the main diagonal (arr[i][j] where i == j). All off-diagonal elements are zero.
Transpose
Swapping rows and columns: element at [i][j] moves to [j][i]. Result is an n×m matrix from an m×n matrix.
Identity Matrix
A square matrix where diagonal = 1, all others = 0. Multiplying any matrix by identity gives the same matrix.
🌍

Real-World Analogy

Matrices are everywhere in the real world wherever data is naturally organized in rows and columns:

1
Spreadsheet / Excel
Each cell is identified by row (number) and column (letter) — exactly like arr[i][j]
2
Digital Image (Pixels)
An image is a 2D matrix of pixels — arr[row][col] gives the color value of that pixel
3
Chess / Game Board
An 8×8 chessboard is a matrix — each square is arr[row][col] holding the piece info
4
Graph Adjacency Matrix
arr[i][j] = 1 means there's an edge from vertex i to j — used to represent graphs
📝

Basic Terminologies

Cell / Element
Each individual value stored at position arr[i][j]. i is the row index, j is the column index.
Row (i)
A horizontal sequence of elements. A matrix with m rows has row indices 0 to m−1.
Column (j)
A vertical sequence of elements. A matrix with n columns has column indices 0 to n−1.
Dimensions (m × n)
m = number of rows, n = number of columns. Total elements = m × n.
Square Matrix
A matrix where m == n (equal rows and columns). Has a main diagonal from [0][0] to [n−1][n−1].
Row-Major Order
Memory layout where all elements of row 0 come first, then row 1, etc. Used in C, C++, Java.

Declaration & Initialization

A matrix must be declared with both its row and column size. Initialization assigns values to all cells at once or using loops.

1
Declaration of Matrix
▼

Declaring a matrix reserves a 2D block of memory. You must specify the number of rows and columns. The syntax varies by language.

  • Specify data type, name, and both dimensions
  • Static declaration: int arr[3][3];
  • Dynamic declaration: use vectors, ArrayList of ArrayLists, or list of lists

Java
Python

C
// Static 2D array declaration in C
int arr[3][3];         // 3x3 matrix of integers
float mat[4][5];      // 4x5 matrix of floats
char grid[10][10];    // 10x10 char grid

// Dynamic allocation
int rows = 3, cols = 3;
int **mat2 = (int**)malloc(rows * sizeof(int*));
for (int i = 0; i < rows; i++)
    mat2[i] = (int*)malloc(cols * sizeof(int));

// Dynamic using vector of vectors

int[][] arr;                         // declare reference
arr = new int[3][3];               // allocate 3x3

Operations on Matrix

A matrix supports several fundamental operations. Each has its own time complexity depending on the size of the matrix (m rows × n columns).

🎯 1. Accessing Elements

Like 1D arrays, matrix elements can be accessed randomly using two indices. arr[i][j] gives the element at row i and column j in O(1) time.

1
Access Element at arr[i][j]
▼
Time Complexity
O(1)
Space Complexity
O(1)

Algorithm:

  • Provide valid row index i (0 ≤ i < m)
  • Provide valid column index j (0 ≤ j < n)
  • Return arr[i][j] directly — O(1) random access

🔁 2. Traversal of Matrix

We visit every element of the matrix using two nested for-loops — the outer loop iterates over rows, the inner loop over columns.

1
Traverse all Elements (Row-wise)
▼
Time Complexity
O(m×n)
Space Complexity
O(1)

Algorithm:

  • Use outer loop for rows: i from 0 to m−1
  • Use inner loop for columns: j from 0 to n−1
  • Process arr[i][j] at each step
  • Total iterations = m × n

🔍 3. Searching in a Matrix

We can search for an element by traversing all cells of the matrix. For an unsorted matrix, this requires checking every element.

1
Search Element in Matrix
▼
Best Case
O(1)
Worst Case
O(m×n)
Space
O(1)

Algorithm:

  • Traverse each row i from 0 to m−1
  • For each row, traverse each column j from 0 to n−1
  • If arr[i][j] == target, return true (or [i, j])
  • If loop ends without finding: return false
C
C++
Java
Python
JavaScript
C
int searchMatrix(int arr[][5], int m, int n, int x) {
    for (int i = 0; i < m; i++)
        for (int j = 0; j < n; j++)
            if (arr[i][j] == x)
                return 1;  // found
    return 0;  // not found
}

// Usage
int x = 8;
int arr[2][5] = {{0,6,8,9,11},{20,22,28,29,31}};
printf("%s\n", searchMatrix(arr, 2, 5, x) ? "YES" : "NO");
C++
bool searchInMatrix(vector<vector<int>>& arr, int x) {
    int m = arr.size(), n = arr[0].size();
    for (int i = 0; i < m; i++)
        for (int j = 0; j < n; j++)
            if (arr[i][j] == x)
                return true;
    return false;
}

int x = 8;
vector<vector<int>> arr = {{0,6,8,9,11},{20,22,28,29,31}};
cout << (searchInMatrix(arr, x) ? "YES" : "NO") << endl;
// Output: YES
Java
boolean searchInMatrix(int[][] arr, int x) {

📊 4. Sorting a Matrix

A matrix can be sorted in two main ways — row-wise (each row sorted independently) or column-wise (each column sorted independently).

1
Sort Matrix Row-wise
▼

Sort each row of the matrix independently. Apply any sorting algorithm (or built-in sort) to each row.

Time Complexity
O(m·n·log n)
Space Complexity
O(1) or O(n)
C
C++
Java
Python
JavaScript
C
#include <stdlib.h>

int cmp(const void* a, const void* b) {
    return (*(int*)a - *(int*)b);
}

void sortRowWise(int arr[][3], int m, int n) {
    for (int i = 0; i < m; i++)
        qsort(arr[i], n, sizeof(int), cmp);
}
C++
#include <algorithm>

void sortRowWise(vector<vector<int>>& arr) {
    for (auto& row : arr)
        sort(row.begin(), row.end());
}

// Usage
vector<vector<int>> mat = {{3,1,4},{9,2,7}};
sortRowWise(mat);
// mat = {{1,3,4},{2,7,9}}
Java
import java.util.Arrays;

void sortRowWise(int[][] arr) {
    for (int[] row : arr)
        Arrays.sort(row);
}

// Usage
int[][] mat = {{3,1,4},{9,2,7}};
sortRowWise(mat);
// mat = {{1,3,4},{2,7,9}}
Python
def sort_row_wise(arr):
    for row in arr:
        row.sort()
    return arr

mat = [[3,1,4],[9,2,7]]
print(sort_row_wise(mat))
# [[1, 3, 4], [2, 7, 9]]
JavaScript
function sortRowWise(arr) {
    return arr.map(row => [...row].sort((a, b) => a - b));
}

const mat = [[3,1,4],[9,2,7]];
console.log(sortRowWise(mat));
// [[1,3,4],[2,7,9]]
2
Sort Matrix Column-wise
▼

Sort each column of the matrix independently. Extract each column into a temporary array, sort it, then put it back.

Time Complexity
O(n·m·log m)
Space Complexity
O(m)
C
C++
Java
Python
JavaScript
C
int cmp(const void*a,const void*b){return*(int*)a-*(int*)b;}

void sortColumnWise(int arr[][3], int m, int n) {
    int col[m];
    for (int j = 0; j < n; j++) {
        for (int i = 0; i < m; i++) col[i] = arr[i][j];
        qsort(col, m, sizeof(int), cmp);
        for (int i = 0; i < m; i++) arr[i][j] = col[i];
    }
}
C++
void sortColumnWise(vector<vector<int>>& arr) {
    int m = arr.size(), n = arr[0].size();
    for (int j = 0; j < n; j++) {
        vector<int> col;
        for (int i = 0; i < m; i++) col.push_back(arr[i][j]);
        sort(col.begin(), col.end());
        for (int i = 0; i < m; i++) arr[i][j] = col[i];
    }
}
Java
import java.util.Arrays;
void sortColumnWise(int[][] arr) {
    int m = arr.length, n = arr[0].length;
    for (int j = 0; j < n; j++) {
        int[] col = new int[m];
        for (int i = 0; i < m; i++) col[i] = arr[i][j];
        Arrays.sort(col);
        for (int i = 0; i < m; i++) arr[i][j] = col[i];
    }
}
Python
def sort_column_wise(arr):
    rows, cols = len(arr), len(arr[0])
    for j in range(cols):
        col = sorted(arr[i][j] for i in range(rows))
        for i in range(rows):
            arr[i][j] = col[i]
    return arr

mat = [[9,1,7],[3,5,2]]
print(sort_column_wise(mat))
# [[3, 1, 2], [9, 5, 7]]
JavaScript
function sortColumnWise(arr) {
    const m = arr.length, n = arr[0].length;
    for (let j = 0; j < n; j++) {
        const col = arr.map(row => row[j]).sort((a,b) => a-b);
        arr.forEach((row, i) => row[j] = col[i]);
    }
    return arr;
}
console.log(sortColumnWise([[9,1,7],[3,5,2]]));
// [[3,1,2],[9,5,7]]
