📋 PSEUDO CODE
INIT: Set search boundaries
low = 0 ; high = n − 1
LOOP: While range is valid
while low ≤ high
MID: Find middle index
mid = low + (high − low) / 2
COMPARE: Check mid element
if arr[mid] == target
FOUND: Return index
return mid
GO RIGHT: Target is larger
if arr[mid] < target → low = mid+1
GO LEFT: Target is smaller
if arr[mid] > target → high = mid−1
NOT FOUND: Range exhausted
return −1
BASE CASE: Range invalid
if low > high: return −1
MID: Find middle index
mid = low + (high − low) / 2
COMPARE: Check mid element
if arr[mid] == target
FOUND: Return index
return mid
RECURSE RIGHT:
return search(arr, mid+1, high)
RECURSE LEFT:
return search(arr, low, mid−1)
NOT FOUND: −1 propagates
return −1 ← all calls unwind
